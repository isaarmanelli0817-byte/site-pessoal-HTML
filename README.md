# Site Pessoal em HTML/CSS

Meu primeiro site desenvolvido no 1º período do curso de Análise e Desenvolvimento de Sistemas.

## Sobre o projeto
Um site pessoal simples com HTML e CSS para praticar estrutura de páginas web, estilização e links.

## O que eu aprendi
- Estrutura básica de HTML5
- Estilização com CSS
- Criação de links e imagens
- Organização de arquivos

## Como visualizar
Basta abrir o arquivo `index.html` no navegador.

## Tecnologias
- HTML5
- CSS3
